POS Allow/Deny Features

Allow/Deny multiple features for POS User

This module allows you to control which user can access how much functionalities in POS:

  • Allow/Deny Payments.
  • Allow/Deny Discount Offerings.
  • Allow/Deny Quantity of Product.
  • Allow/Deny Editing Product Price.
  • Allow/Deny Removing order lines.

To access these features, Go to Settings -> Users, Open User and click on "Point of Sale" tab.

How it works

Allow/Deny desired fields as Admin.

Here We have denied Payments and Edit price from POS orders.


These changes we have made are reflected into POS. Payments and Edit Price buttons are disabled.
